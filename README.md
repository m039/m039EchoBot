## Installation

```bash
cd <project-directory>

# Install dependencies.
composer install

# Add configuration parameters.
echo "TOKEN=..." > .config.ini
echo "WEBHOOK_SECRET=..." >> .config.ini

# Set the webhook url.
composer setup-webhook --set "..."

```
